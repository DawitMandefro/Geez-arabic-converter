# Title

 Geez-arabic-converter

 ## project description

 This repository contains a system that changes arabic number into geez ones and viceversa.

 ## problem statment

 its making a system that converts arabic or geez number into geez number and arabic number respectivley.It accepts geez or arabic number from user and changes it into respective one.

 ## project objective

 build a solution that efficiently handles the problem

 ## Repository structure 

 /docs ---problem_analysis.md   -----> stores analysis document 
 /docs ---geez_number_rule.md   -----> stores rules used in geez number

/src  ---arabic_to_geez.py           ----->  implementaton of arabic to geez conversion in python

/src  ---geez_to_arabic.py           ----->  implementaton of geez to arabic conversion in python

/src  ---converter.py           ----->  implementaton of main entry point

/test_case ---testcases.py     -----> contains tests to assure Quality
README.md                      -----> contain general overview of system
